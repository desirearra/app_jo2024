# Tests d'intégration du modèle utilisateur

Ce dossier contient tous les tests d'intégration pour le modèle utilisateur (CRUD, auth, etc.).
Chaque fichier de test couvre une route ou une fonctionnalité spécifique.

**Exemples :**

- `users.me.test.ts` : GET /api/users/me
- `users.list.test.ts` : GET /api/users
- `users.getById.test.ts` : GET /api/users/:id
- `users.update.test.ts` : PUT /api/users/:id
- `users.delete.test.ts` : DELETE /api/users/:id

> Ce fichier sert uniquement de documentation et d'organisation pour les tests du modèle utilisateur.
